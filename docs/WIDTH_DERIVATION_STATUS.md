# Status of the width-law derivation (2026-07-25)

Target: derive s ≈ 0.012·V (manuscript eq. 3) from the generative account.

## What is PROVEN (computationally)

1. **The mixture generates the law.** Simulating each corpus's fitted
   zero-truncated PLN mixture through the observation model and fitting the
   canonical 9-parameter erf-gate model reproduces the measured seam width at
   ratio ≈ 1.0 on 24/25 corpora — 16 at the raw ML basin (f13), 8 more once
   the basin is selected by cross-size prediction error (f13b), a criterion
   that never sees rank curves or widths. Kant under extended search (f13c-K).
2. **Basin-criterion convergence.** Cross-size V/c prediction and seam width —
   two independent observables — select the same likelihood basin (ΔNLL 0–11
   from ML). The physical basin is identified by prediction, not likelihood.
3. **The depth dimension is load-bearing.** Basins that match the single-depth
   histogram but not its thinned versions get the width wrong; basins that
   match both get it right. Consistent with §3.4 (c = sampling depth).

## What was RULED OUT (f13c)

The natural closed-form candidate — the "ambiguous mass"
A = N_total·∫_band (mixture density)·(1−e^(−λ)) dx over the posterior-share
handover band — is NOT the width. A/s scatters 0.14–76 (corr 0.51, median
3.2) across the very basins whose *simulated* widths are exact. The width is
not a λ-space band mass; it is created by the composition
(observation floor) ∘ (rank transform) ∘ (erf-fit operator), and no shortcut
that stops in λ-space captures it. Band-definition sensitivity (crossing-pair
vs share-band give A differing by ~6× on the same corpus) confirms the
λ-space quantity is not the invariant.

## The remaining analytic route (open; Paper-2 scope)

Compute the EXPECTED rank curve from the mixture analytically —
r(f) = N_total·P(n ≥ f) with n ~ zero-truncated PLN — and characterize the
erf-gate width of the resulting deterministic log-rank curve near the
handover, asymptotically in the fitted-parameter regime (π_H ~ 1–4%,
gap ~ 4.5–7, σ_T ~ 1.5–2.1). The question "why 1.2%" then becomes: why do
physical-basin parameter combinations map to a stable fitted-width fraction
under this operator? The operator's fitting step (what the 9-parameter model
*chooses* to call the transition on the exact curve) must be part of the
statement — the f13c negative shows it cannot be dropped.
